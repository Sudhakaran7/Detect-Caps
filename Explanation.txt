Given a word, you need to judge whether the usage of capitals in it is right or not.
We define the usage of capitals in a word to be right when one of the following cases holds:
All letters in this word are capitals, like "USA".
All letters in this word are not capitals, like "Amazon".
Only the first letter in this word is capital, like "Paytm".
Otherwise, we define that this word doesn't use capitals in a right way.

Input description:
A string S as input.

Output Description:
Print yes or no

Sample Input:
USA

Sample Output:
yes

Explanantion:
In the given string the upper case is placed and used correctly.

Sasmple Input:
Amazon

Sample Output:
yes

Sample Input:
yeS

Sample Output:
no

Sample Input:
pAytM

Sample Output:
no

Sample Input:
dog

Sample Output:
no

Sample Input:
No

Sample Output:
yes
